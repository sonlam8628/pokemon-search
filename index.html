Pokemon 卡牌查詢

適合部署到 GitHub Pages 的靜態網站版本。

搜尋 Pokemon 名稱，或用 set id + 卡號精準找卡圖

這個版本針對正式靜態部署整理成單一 index.html，不依賴本機伺服器。你可以直接放到 GitHub Pages，上線後用公開網址查詢卡片資料與圖片。

部署前後都適用的使用提示
  • 名稱模式可先試 Pikachu、Eevee、Charizard。
  • 精準模式可輸入像 base1 + 4。
  • 卡片可點擊放大，方便確認圖案與編號。
  • 若個別圖片載入失敗，卡面區會顯示備援提示，不會整張版面壞掉。
建議上 GitHub Pages 後再測試圖片載入，靜態網站公開網址通常比沙盒預覽更接近真實上線環境。
